# Phonebook

Simple phonebook application. Can add contact by supplying their name and phone number. You are also able to delete a contact or search for them by their name.

FrontEnd
- BootStrap
- ReactJS
- Axios

BackEnd
- Node.JS
- Express
- MongoDB
